## :material-target: Was in 10/5 Minuten realistisch ist

!!! tip "Realistisch"
    - **Ein** klar abgegrenztes Problem/Szenario vollständig darstellen
    - Strukturiertes Vorgehen zeigen (Problem → Analyse → Lösung → Ergebnis)
    - 7-9 Folien mit je 1-2 Minuten Redezeit
    - Eine Entscheidung mit Begründung („Warum Option A statt B?")
    - Ergebnis mit messbarem Nutzen zeigen

!!! warning "Nicht realistisch"
    - Mehrere komplexe Themen abdecken
    - Tiefe technische Deep-Dives mit Live-Demos
    - Vollständige Prozessdokumentation vorlesen
    - Jedes Detail der Implementierung erklären
    - Mehr als 10-12 Folien sinnvoll besprechen
